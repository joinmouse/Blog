## 单态
这是设计模式中最基础的一个，也是最常用的一个模式。在面向对象中我们都知道首先创建一个类，然后可生多个对象来调用类的属性和方法，那这里的**单态就是指的只让某个类仅仅只生成一个对象**。

## 使用php语言来看是如何实现单态的
我们首先来分析下，如何才可以使用让一个类只有一个对像？这里我们不妨先让这个类不能创建对象，将构造方法私有化(PHP中使用private修饰符)

```
class Person {
    private function __construct(){}
}
```
那么可能有人要问了，你这样把构造方法私有化了，那应该如何生成一个对象啊
在类中我们知道可以创建一个静态方法，我们可以在这个静态方法中返回一个新的对象，这样就可以达到不使用构造函数来创建对象的目的啊!

```
class Person {
    static $object = null

    private function __construct(){}
    static function getObject(){
        self::$object = new Person
        return self::$object
    }
}
$person = Person::getObject()
$person1 = Person::getObject()
$person2 = Person::getObject()
```

这样就可以了吗，如果我们细心的话test会发现当创建对象的时候，还是可以在栈中生成多个对象啊，这个时候我们将上面的代码添加一个条件语句，就可以达到无论多次调用都仅仅只创建了一个对象的目的
```
class Person {
	static $object = null;
	
	private function __construct(){}
	
	static function getObject(){
		//若第一次调用时，没有对象则创建；以后调用，直接使用第一次调用的对象
		if(is_null(self::$object))
			self::$object = new Person;
		return self::$object;
	}
	
	function say() {
		echo "aaaaaaa<br>";
	}
	
	//析构方,在脚本执行后自动执行，这里用来判断创建了几次对象
	function __destruct() {
		echo "#######<br>";
	}
}
```
